# PS Cheese Cafe — Luxury Website Experience

A premium, highly interactive, and responsive web experience for **PS Cheese Cafe by Roastery** (Hyderabad). Built using a modern frontend stack with a focus on immersive animations, bespoke typography, and a tailored design system.

Live Production URL: **[https://cafepscheese.vercel.app](https://cafepscheese.vercel.app)**

---

## ✨ Features

- **Sun-Drenched Glass Roof Atrium**: Clean, immersive visual branding showcasing the cafe's signature glass ceiling architecture.
- **Hyderabad's First In-House Cheese Factory**: Explains the artisan cheesemaking process (fresh mozzarella, ricotta, cream cheese) with a premium Bento-style Highlights grid.
- **Asymmetric Photo Gallery**: A custom Bento Grid displaying curated spot-focused photography under natural light, with smooth spring hover-lifts, scale transitions, and a clean fullscreen lightbox viewer.
- **Specialty Coffee powered by Roastery**: Dedicated Coffee Experience section detailing origin-focused brewing, pour-overs, and cold brews.
- **Bespoke Signature Favorites**: Highlights gourmet food items with parallax scroll reveals and smooth visual cards.
- **WhatsApp Table Reservation**: Integrated reservation routing using clean CTA buttons.

---

## 🛠️ Technology Stack

- **Core**: React 19, TypeScript, Vite
- **Styling**: TailwindCSS, Vanilla CSS (custom typography and variables)
- **Animations**: Framer Motion (for smooth physics-based transitions, spring highlights, and page reveals)
- **Deployment**: Vercel (Production Build & Cache Optimization)

---

## 🚀 Local Development

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/mohanx24/ps-cheese-cafe.git
   cd ps-cheese-cafe
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Start Dev Server**:
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` in your browser.

4. **Build for Production**:
   ```bash
   npm run build
   ```
